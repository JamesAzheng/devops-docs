# 插件管理相关变量

```sql
#动态加载插件，如果值为NO 则不能使用如--plugin-load在服务器启动时加载插件，或INSTALL PLUGIN在运行时加载插件的语句
mysql> select @@have_dynamic_loading;
+------------------------+
| @@have_dynamic_loading |
+------------------------+
| YES                    | #YES表示支持动态加载，NO反之
+------------------------+


#插件存放目录
mysql> select @@plugin_dir;
+------------------------------------------------------------+
| @@plugin_dir                                               |
+------------------------------------------------------------+
| /usr/local/mysql-8.0.26-linux-glibc2.12-x86_64/lib/plugin/ |
+------------------------------------------------------------+

```
