---
title: ufw
---

在默认情况下，Ubuntu操作系统是没有启用防火墙的。然而，Ubuntu使用的是Uncomplicated Firewall（UFW）作为防火墙管理工具，它提供了一种简化配置的方式来管理防火墙规则。

当您安装Ubuntu时，默认情况下，UFW处于禁用状态，不会阻止任何网络连接。您可以使用以下命令来检查UFW的状态：

```
sudo ufw status
```

如果输出结果显示"Status: inactive"，则表示UFW当前处于禁用状态。

如果您需要启用防火墙并配置规则来限制网络访问，您可以使用以下命令启用UFW：

```
sudo ufw enable
```

一旦UFW被启用，它将开始阻止未经授权的网络连接，并根据您的配置规则进行过滤。

请注意，如果您使用其他防火墙工具，例如iptables，那么默认情况下Ubuntu可能不会使用UFW。在这种情况下，您需要手动配置和管理防火墙规则。

建议在配置防火墙之前仔细考虑，并确保了解您的网络需求和安全要求，以免意外地阻止了所需的网络连接。
