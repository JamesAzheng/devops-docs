# 网卡概述

- 处于OSI七层参考模型的 数据链路层

- 进行串行/并行转换
- 数据缓存
- 在计算机操作系统中安装设备驱动程序
- 实现以太网协议
- 网卡中存有一个唯一的mac地址（设备的唯一标识）



# X710

X710是英特尔推出的网卡产品系列，特别是面向数据中心和云计算等高密度、高吞吐量服务器应用场景。以下是该网卡系列的一些详细信息：

1. 网卡类型：支持万兆以太网（10GbE）和40GbE网络连接。
2. 特征：X710网卡支持SR-IOV、VMDq、VxLAN、DCB、PCIe Gen 3.0和RDMA等高级网络协议和特征，以提供更快的数据传输速率、更低的网络延迟和更高的网络质量。
3. 芯片架构：X710系列网卡采用英特尔的Fortville芯片，该芯片通过在硬件级别实现网络协议加速、分组过滤、流量管理和负载平衡等功能，提供更高密度、更快速的数据中心应用。
4. 适用场景：X710网卡适用于高度虚拟化的数据中心、云计算、存储和网络应用等场景。该网卡系列具有高性能、高密度、高可靠性和高安全性等特点，是满足数据中心网络需要的重要组成部分。

需要注意的是，为了发挥X710系列网卡的最佳性能，您需要安装最新的英特尔驱动程序和固件，并进行适当的配置。
